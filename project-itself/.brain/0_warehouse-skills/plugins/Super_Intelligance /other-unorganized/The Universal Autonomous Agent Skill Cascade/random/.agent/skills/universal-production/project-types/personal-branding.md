# Personal Branding Website Skill

Use this skill for personal brands, influencers, coaches, speakers, authors.

## Required Materials Checklist

### Personal Brand
- [ ] Full name
- [ ] Professional title/tagline
- [ ] Brand colors (if established)
- [ ] Personal logo or wordmark

### Personal Content
- [ ] High-quality headshot
- [ ] Multiple lifestyle photos
- [ ] Video introduction (optional)
- [ ] Bio (short + long version)
- [ ] Personal story

### Expertise
- [ ] Area of expertise
- [ ] Key topics/pillars
- [ ] Speaking topics (if speaker)
- [ ] Books/courses (if any)

### Social Proof
- [ ] Follower counts
- [ ] Media mentions
- [ ] Client testimonials
- [ ] Speaking engagements
- [ ] Awards/recognition

### Content
- [ ] Blog posts or articles
- [ ] Newsletter signup
- [ ] Lead magnet (free download)
- [ ] Podcast/YouTube links

## If Materials Missing

### Missing Headshot Prompt
```
"Professional personal branding headshot of a [gender] [profession],
confident pose, warm smile, modern background,
suitable for speaker/author/coach website"
```

### Missing Lifestyle Photos
```
"Personal brand lifestyle photo of a [profession] speaking at conference,
engaging with audience, professional setting, action shot"
```

### Missing Book Cover
```
"Book cover mockup for '[book title]' by [author name],
modern professional design, [topic] theme, bestseller look"
```

## Recommended Structure

```
/                   → Hero + key message + opt-in
/about              → Story + credentials
/speaking           → Topics + past events + booking
/coaching           → Programs + pricing + apply
/books              → Publications with purchase links
/podcast            → Episodes + subscribe
/blog               → Content hub
/contact            → Booking + inquiry form
```

## Personal Brand Elements

### Hero Section Must Have
1. Powerful headline (value proposition)
2. Professional photo
3. Social proof (as seen on, followers)
4. Clear CTA (opt-in or book call)

### Trust Building
- "As Seen On" logos
- Testimonial video
- Client results
- Media features
- Certifications

## Lead Generation

### Primary Opt-In
- Free resource (ebook, checklist, course)
- Email capture
- Thank you sequence

### Secondary CTAs
- Book a call
- Join newsletter
- Follow on social

## Design Guidelines

- Personality-driven design
- Consistent with social media aesthetic
- Large photos
- Bold typography
- Story-focused
- Mobile-optimized
