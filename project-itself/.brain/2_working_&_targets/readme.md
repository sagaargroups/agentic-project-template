# 📝 Working Sessions & Docs

> **Your project's day-to-day memory.**

## What Lives Here
This is where you store the **active, in-progress work** of your project. Every significant AI-human working session, design discussion, or documentation draft gets saved here.

### Examples of What to Store
- 💬 AI-Human chat transcripts from building sessions
- 📄 Wiki and documentation drafts
- 🏗️ Architecture decision records (ADRs)
- 📐 Diagrams and flowcharts
- 📋 Scratchpads and brainstorm notes

## How to Use This Folder
1. When you start a new working session with your AI, save the transcript as `session_YYYY-MM-DD.md`
2. If a session produces a reusable skill or tool, move it to the appropriate warehouse
3. Keep transcripts here as historical reference — your AI will read them for context

## Pro Tip
Your AI agent reads this folder to understand *why* past decisions were made. The more detailed your sessions, the smarter your AI becomes over time.
