# 📋 Tasks & Daily Logs — AI Context

> **Folder Type:** Personal Project Space
> **Audience:** AI Agents

## Purpose
The **strategic backlog and activity ledger**. This folder holds task checklists, feature requests, bug trackers, and daily contribution logs that track the project's progress over time.

## Allowed Contents
- Markdown task checklists (`- [ ]`, `- [x]`, `- [/]` for in-progress)
- Feature request outlines
- Prioritized bug reports
- Daily contribution logs (what was built, fixed, or researched each day)
- Sprint planning documents and epics

## Operational Rules
- **Prioritization:** Tasks must be clearly prioritized (🔴 High, 🟡 Medium, 🟢 Low). The human founder dictates priority.
- **Status Tracking:** Update task statuses immediately upon completion to keep AI agents synchronized.
- **Daily Logs:** Append a daily entry summarizing what was accomplished, what blockers exist, and what's next.

## Daily Log Format
```markdown
## [YYYY-MM-DD]
### ✅ Completed
- [What was done]

### 🚧 In Progress
- [What's being worked on]

### 🔜 Next
- [What's planned next]

### 🧱 Blockers
- [Any blockers or questions]
```

## AI Guardrails
- **NEVER** start working on a task from this list without explicit authorization from the human founder. This is a queue, not a command line.
- **ALWAYS** check this folder at the start of every session for high-priority pending items.
- **ALWAYS** log your contributions after completing significant work.
