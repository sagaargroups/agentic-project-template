# 🎨 Visual Identity Guide

> Update this with YOUR brand colors, fonts, and style preferences.
> The image-creator skill reads this before generating any visuals.

## Colors
| Name | Hex | Usage |
|---|---|---|
| Primary | #000000 | Main brand color |
| Secondary | #000000 | Accents, CTAs |
| Background | #FFFFFF | Page/card backgrounds |
| Text | #1A1A1A | Body text |
| Accent | #000000 | Highlights, links |

## Typography
| Usage | Font | Weight |
|---|---|---|
| Headings | Inter | Bold (700) |
| Body | Inter | Regular (400) |
| Code | JetBrains Mono | Regular (400) |

## Image Style
- **Photography:** [Bright and airy / Dark and moody / Natural]
- **Illustrations:** [Flat / 3D / Hand-drawn / Isometric]
- **Icons:** [Outline / Filled / Colored]
- **Backgrounds:** [Gradient / Solid / Textured]

## Logo Usage
- Minimum size: 32px height
- Clear space: equal to the height of the logo icon
- Never stretch, rotate, or add effects
- Dark backgrounds: use white version
- Light backgrounds: use dark version
