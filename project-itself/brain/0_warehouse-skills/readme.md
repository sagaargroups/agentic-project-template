# 📦 Warehouse: Skills & Automations

> **Community Warehouse** — grows with every contribution.

## What Lives Here
Reusable AI agent skills, bundled plugins, and third-party integrations.

| Subfolder | Purpose |
|-----------|---------|
| `skills/` | 🎯 Individual single-purpose skills |
| `plugins/` | 🔌 Bundled multi-skill packages for big workflows |
| `third-party/` | 🌐 Skills cloned from external sources |

## How to Contribute
1. Pick the right subfolder (`skills/`, `plugins/`, or `third-party/`)
2. Create a folder with a kebab-case name
3. Add a `SKILL.md` (for skills) or `README.md` (for plugins) following the standard format
4. Submit a PR — see [CONTRIBUTING.md](/CONTRIBUTING.md)
