# 📝 Content Templates & Formulas

## Hook Formulas (First Line)
These are proven opening lines that stop the scroll:

### Question Hook
```
"Are you still [doing painful thing] in [current year]?"
```

### Contrarian Hook
```
"Stop [common advice]. Here's what actually works:"
```

### Number Hook
```
"[Number] [things] that [benefit] in [timeframe]"
```

### Story Hook
```
"[Time] ago, I was [relatable struggle]. Then I discovered [solution]."
```

### Challenge Hook
```
"Try this for [timeframe] and watch [result] happen:"
```

## Content Frameworks

### PAS (Problem → Agitate → Solution)
```
[Describe the problem]
[Make it feel urgent/painful]
[Present your solution]
```

### AIDA (Attention → Interest → Desire → Action)
```
[Grab attention with hook]
[Build interest with facts/story]
[Create desire with benefits]
[Call to action]
```

### BAB (Before → After → Bridge)
```
[Paint the "before" picture]
[Show the "after" result]
[Bridge: here's how to get there]
```

### Value Ladder
```
Tweet → Thread → Blog → Newsletter → Course → Product
(Each step gives more depth on the same topic)
```
