# 🔧 Warehouse: Helper Apps & Tools

> **Community Warehouse** — grows with every contribution.

## What Lives Here
Standalone utilities, bundled tool suites, and third-party integrations.

| Subfolder | Purpose |
|-----------|---------|
| `tools/` | 🔧 Individual single-purpose utilities |
| `plugins/` | 🔌 Bundled multi-tool suites for complete workflows |
| `third-party/` | 🌐 Tools cloned from external sources |

## How to Contribute
1. Pick the right subfolder (`tools/`, `plugins/`, or `third-party/`)
2. Create a folder with a kebab-case name
3. Add a `README.md` explaining what it does and how to run it
4. Include `.env.example` if config is needed
5. Submit a PR — see [CONTRIBUTING.md](/CONTRIBUTING.md)
