# 🔌 Plugins (Bundled Tools)

Tool bundles — multiple related utilities packaged together for a complete workflow.

```
plugins/
└── my-tool-suite/
    ├── README.md         # Overview of the suite
    ├── tool-a/           # First tool
    ├── tool-b/           # Second tool
    └── shared/           # Shared utilities between tools (optional)
```

**Example use cases:**
- A "Database Suite" bundling migration + seeding + backup tools
- A "Content Pipeline" bundling scraper + formatter + publisher tools
