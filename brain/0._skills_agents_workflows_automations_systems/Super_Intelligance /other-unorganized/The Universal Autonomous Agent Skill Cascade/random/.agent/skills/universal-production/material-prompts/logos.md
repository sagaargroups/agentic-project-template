# Logo Generation Prompts

## Minimalist Logo

```
Professional minimalist logo for [company name], a [industry] company.
Style: clean, modern, geometric, memorable.
Colors: [primary color] and [secondary color].
Suitable for web, mobile app, and business cards.
White background, vector-style clean edges.
```

## Wordmark Logo

```
Elegant wordmark logo typography of "[company name]".
Style: modern sans-serif, professional, unique letterforms.
Color: [brand color] on white background.
Suitable for website header and business documents.
```

## Icon Logo (App Icon Style)

```
App icon logo for [company name], representing [what company does].
Style: flat design, simple shape, scalable.
Colors: [brand colors], suitable for dark and light backgrounds.
Square format, rounded corners, clean and recognizable.
```

## Abstract Logo

```
Abstract geometric logo representing [concept: growth/connection/speed/trust].
Style: modern, professional, unique.
Colors: gradient from [color1] to [color2].
Suitable for tech/startup company.
White background, vector-style.
```

## Mascot Logo

```
Friendly mascot character for [company name], a [industry] brand.
Character: [animal or object] with [personality trait].
Style: modern illustration, approachable, memorable.
Colors: [brand colors].
Suitable for logo and marketing materials.
```

## Variables to Replace

| Variable | Example |
|----------|---------|
| [company name] | CloudFetch |
| [industry] | cloud storage |
| [primary color] | blue |
| [secondary color] | white |
| [concept] | growth, trust, speed |

## Negative Prompts (Add to Avoid)

```
--no text
--no gradients (if you want flat)
--no complex details
--no photorealistic
--no 3D effects
```
