## 🎯 Core Capabilities

### 1. Knowledge Adaptive

| Intermediate | Suggests best practices |

### 2. Context Persistence
- ✅ Remembers across chat sessions
- ✅ Tracks work history
- ✅ Resumes from where you left off
- ✅ Never lose progress

### 3. Production Ready Out-of-Box
Every project automatically gets:
- ✅ Loading states on buttons
- ✅ Error handling
- ✅ Dark/Light mode support
- ✅ Mobile responsive
- ✅ SEO basics
- ✅ Accessibility compliance
- ✅ Security best practices

### 4. Framework Support (Dynamic)

> Frameworks are loaded from `registry.md` - not hardcoded!

**Currently Available:**
- Next.js (full-stack, SEO, ecommerce)
- React + Vite (SPA, dashboards)
- Vue 3 (progressive, quick projects)
- Express (API-only, backend)

**Coming Soon:** Svelte, Astro, Nuxt 3

> To add new framework: Create folder in `framework-templates/`, update `registry.md`

### 5. Project Types (Dynamic)

> Project types are loaded from `registry.md` - easily extensible!

**Currently Available:**
- Portfolio websites
- Business websites
- Local business sites
- Ecommerce stores
- Personal branding

**Coming Soon:** SaaS apps, Blogs, Documentation sites

> To add new project type: Create `.md` file in `project-types/`, update `registry.md`

### 6. Material Generation
Missing assets? Get AI prompts for:
- 🎨 Logos
- 📸 Professional headshots
- 🖼️ Product photos
- 🎯 Icons

## 🛡️ Fail-Safe Features

| Feature | What It Does |
|---------|--------------|
| **Safe Writes** | Writes to temp, then moves (can't corrupt) |
| **Auto Backups** | Before any destructive action |
| **Checkpoints** | Every 5 changes |
| **Rollback** | Can undo to any checkpoint |
| **Graceful Errors** | Never leaves project broken |
