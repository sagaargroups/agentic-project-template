# BRAND IDENTITY MASTER
### The Foundation — Build This First. Everything Else Plugs Into It.

## THE LAW
> A brand only exists when a clear, consistent signal enters a human mind repeatedly.
> Identity is that signal. Build it once. Use it forever.

---

# PART 1 — VISUAL IDENTITY

## 1. LOGO KIT
| Asset | Format Needed | Use |
|---|---|---|
| Primary logo (full) | SVG, PNG, PDF | Main usage everywhere |
| Secondary logo (horizontal) | SVG, PNG | Headers, banners |
| Icon only | SVG, PNG, ICO | Favicon, app icon, profile pic |
| Dark version | SVG, PNG | Light backgrounds |
| Light version | SVG, PNG | Dark backgrounds |
| Monochrome | SVG, PNG | Legal, print, stamp |

**Rules to Define:**
- Minimum size (never smaller than X px)
- Clear space around logo (never crowd it)
- What you can never do to the logo (stretch, recolor, drop shadow)

---

## 2. COLOR SYSTEM
| Role | Count | Use |
|---|---|---|
| Primary color | 1 | The brand color. Dominant everywhere |
| Secondary color | 1–2 | Support, accents |
| Neutral colors | 3–4 | Backgrounds, text, dividers |
| Semantic colors | 3 | Success (green), Warning (yellow), Error (red) |

**For Every Color Define:**
- HEX (web/digital)
- RGB (screen)
- CMYK (print)
- HSL (design tools)

---

## 3. TYPOGRAPHY SYSTEM
| Role | Use |
|---|---|
| Display / Headline font | Hero text, titles, big statements |
| Body font | Paragraphs, descriptions, captions |
| Mono font (optional) | Code, data, technical brands |

**For Each Font Define:**
- Size scale: H1 → H2 → H3 → Body → Caption → Label
- Weight variations: Light / Regular / Medium / Bold / Black
- Line height rules
- Letter spacing rules
- Web fallback stack

**File Formats to Have:**
- OTF / TTF (desktop, print)
- WOFF / WOFF2 (web)

---

## 4. IMAGERY STYLE
| Element | Define |
|---|---|
| Photography style | Real people? Flat lay? Abstract? Dark? Light? |
| Illustration style | Flat? 3D? Line art? Hand drawn? |
| Icon style | Outlined? Filled? Rounded? Sharp? |
| Pattern / texture | Does brand use patterns? What kind? |
| Image filters | Consistent editing style across all photos |

# PART 2 — BRAND VOICE IDENTITY

## 5. BRAND PERSONALITY
- If this brand was a person — who are they?
- Age feel (young/mature?)
- Energy (calm/bold/playful/serious?)
- 3 personality words (e.g. Sharp. Honest. Ambitious.)

---

## 6. TONE OF VOICE
| Channel | Tone | Example |
|---|---|---|
| Website | Professional + clear | "We build what matters." |
| Social Media | Human + direct | "Here's what nobody tells you about..." |
| Email | Personal + warm | "Hey [name], quick thing —" |
| Ads | Bold + urgent | "The last tool you'll need." |
| Legal/Docs | Formal + precise | "As per the agreement dated..." |

---

## 7. BRAND VOICE RULES
- Words we always use
- Words we never use
- Sentence length preference (short/long)
- Do we use humor? How much?
- Do we use slang? Which kind?
- How do we handle criticism publicly?

---

## 8. TAGLINE + CORE MESSAGE
| Element | Your Answer |
|---|---|
| Tagline | One line. Memorable. Forever. |
| Mission | What we do and for whom |
| Vision | Where we are going |
| Value proposition | Why us over anyone else |
| Elevator pitch | 2 sentences. Anyone understands instantly. |
