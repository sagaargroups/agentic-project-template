## 4. Do's and Don'ts

| ✅ DO | ❌ DON'T |
|---|---|
| Speak directly to the reader | Use corporate jargon |
| Be specific with claims | Make vague promises |
| Show, don't just tell | Overuse superlatives |
| Acknowledge mistakes openly | Deflect blame |
| Use the brand's vocabulary | Use competitor's language |
| Keep it simple | Over-explain |

---

## 5. Handling Criticism

| Situation | Response Approach |
|---|---|
| Public complaint (social) | Acknowledge → Apologize → Fix → Follow up |
| Negative review | Thank → Understand → Offer resolution |
| Trolling | Ignore or respond with grace — never engage emotionally |
| Valid criticism | Own it publicly. Fix it visibly. |
| Feature request | Thank → Log → Communicate timeline honestly |

## 6. Content Formatting Rules

| Element | Rule |
|---|---|
| Headlines | {{DISPLAY_FONT}}, Bold, short (5–8 words ideal) |
| Subheadings | Clear, descriptive, not clever |
| Body text | Scannable. Use bullets when possible. |
| CTAs | Action verbs. "Get started" not "Click here" |
| Links | Descriptive text, never "click here" |
| Numbers | Use digits, not words (write "5" not "five") |
