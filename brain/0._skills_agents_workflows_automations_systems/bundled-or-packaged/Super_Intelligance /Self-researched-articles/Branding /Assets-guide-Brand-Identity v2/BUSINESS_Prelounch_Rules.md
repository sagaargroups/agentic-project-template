## 🤖 AI INSTRUCTIONS — READ THIS FIRST

> **You are an AI branding assistant.** Before creating any asset folder or file, you MUST collect all required brand information from the user by asking structured questions. Never assume. Never use placeholder brand names in final output.

### STEP 1 — ASK THE USER THESE QUESTIONS FIRST

Before touching any file, introduce yourself and collect the following from the user. Ask them one section at a time so they are not overwhelmed:

---

**🔵 SECTION A — CORE IDENTITY**
1. What is the **brand name**? (exact casing matters — e.g. "Golu Snacks", "CloudFetch")
2. What is the **brand name in lowercase**? (for filenames/URLs — e.g. "golu-snacks", "cloudfetch")
3. What is the **tagline**? (one line, memorable — e.g. "The Home For Applications", "Taste Ka Asli Maza")
4. What is the **domain**? (e.g. "golusnacks.com", "cloudfetch.tech")
5. What is the **support email**? (e.g. "support@golusnacks.com")
6. What is the **business type**? (SaaS / D2C / Food & Beverage / Agency / Other)

---

**🟠 SECTION B — VISUAL IDENTITY**
7. What is the **primary brand color**? (HEX — e.g. #F97316 orange, #E63946 red)
8. What are the **secondary colors**? (1–2 HEX values)
9. What **neutral colors** to use? (backgrounds, text — e.g. #0F172A dark, #F8FAFC light)
10. What is the **headline / display font**? (e.g. Inter, Poppins, Playfair Display)
11. What is the **body font**? (e.g. Inter, Roboto, Open Sans)
12. What is the **logo concept / visual metaphor**? (e.g. "cloud + download arrow", "spice bowl with steam")
13. What is the **photography style**? (Real people / Flat lay / Abstract / Minimal / Bold)
14. What is the **illustration style**? (Flat / 3D / Line art / Hand drawn / None)
15. What is the **icon style**? (Outlined / Filled / Rounded / Sharp)

---

**🟢 SECTION C — BRAND VOICE**
16. If this brand was a person — who are they? (age, energy, personality)
17. **3 personality words** (e.g. "Sharp. Honest. Ambitious." or "Warm. Playful. Bold.")
18. What **words does the brand always use**?
19. What **words does the brand never use**?
20. Does the brand use **humor**? How much? (Never / Subtle / Often / Core identity)
21. Does the brand use **slang/casual language**? What kind?

---

**🟣 SECTION D — MESSAGING**
22. What is the **mission statement**? (What we do and for whom)
23. What is the **vision statement**? (Where we are going)
24. What is the **value proposition**? (Why us over anyone else)
25. What is the **elevator pitch**? (2 sentences — anyone understands instantly)

---

**🔴 SECTION E — DIGITAL PRESENCE**
26. Which **social platforms** are priority? (Instagram, Facebook, LinkedIn, Twitter/X, YouTube, WhatsApp, TikTok, etc.)
27. What **country/region** for legal registration? (India / US / Global)
28. Is **MSME / GST / Trademark** needed? (Yes / No / Already done)

# PART 1 — VISUAL IDENTITY

## 1. LOGO KIT
| Asset | Format Needed | Use |
|---|---|---|
| Primary logo (full) | SVG, PNG, PDF | Main usage everywhere |
| Secondary logo (horizontal) | SVG, PNG | Headers, banners |
| Icon only | SVG, PNG, ICO | Favicon, app icon, profile pic |
| Dark version | SVG, PNG | Light backgrounds |
| Light version | SVG, PNG | Dark backgrounds |
| Monochrome | SVG, PNG | Legal, print, stamp |

**Rules to Define:**
- Minimum size (never smaller than X px)
- Clear space around logo (never crowd it)
- What you can never do to the logo (stretch, recolor, drop shadow)

---

## 2. COLOR SYSTEM
| Role | Count | Use |
|---|---|---|
| Primary color | 1 | The brand color. Dominant everywhere |
| Secondary color | 1–2 | Support, accents |
| Neutral colors | 3–4 | Backgrounds, text, dividers |
| Semantic colors | 3 | Success (green), Warning (yellow), Error (red) |

**For Every Color Define:**
- HEX (web/digital)
- RGB (screen)
- CMYK (print)
- HSL (design tools)

---

## 3. TYPOGRAPHY SYSTEM
| Role | Use |
|---|---|
| Display / Headline font | Hero text, titles, big statements |
| Body font | Paragraphs, descriptions, captions |
| Mono font (optional) | Code, data, technical brands |

**For Each Font Define:**
- Size scale: H1 → H2 → H3 → Body → Caption → Label
- Weight variations: Light / Regular / Medium / Bold / Black
- Line height rules
- Letter spacing rules
- Web fallback stack

**File Formats to Have:**
- OTF / TTF (desktop, print)
- WOFF / WOFF2 (web)

---

## 4. IMAGERY STYLE
| Element | Define |
|---|---|
| Photography style | Real people? Flat lay? Abstract? Dark? Light? |
| Illustration style | Flat? 3D? Line art? Hand drawn? |
| Icon style | Outlined? Filled? Rounded? Sharp? |
| Pattern / texture | Does brand use patterns? What kind? |
| Image filters | Consistent editing style across all photos |
